# Erntemanagement

Das Erntemanagement verfolgt Erntebatches, Qualitätsbewertungen und Ertragskennzahlen. Die Karenzzeit-Prüfung (IPM) blockiert Ernten bei aktiven Pflanzenschutzbehandlungen.
