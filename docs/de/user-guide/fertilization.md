# Dünge-Logik

Die Dünge-Logik von Kamerplanter plant Nährstoffgaben mit EC-Budget-Kalkulation, Mischfolge-Validierung und Spülprotokollen. Elektrische Leitfähigkeit (EC) misst die Nährstoffkonzentration in der Lösung.

!!! warning "Wichtige Mischfolge"
    CalMag immer **vor** Sulfaten mischen, um Ausfällungen zu verhindern.
