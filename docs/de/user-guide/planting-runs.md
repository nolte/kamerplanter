# Pflanzdurchläufe

Ein Pflanzdurchlauf (Planting Run) gruppiert mehrere Pflanzeninstanzen für gemeinsames Lifecycle-Tracking. Batch-Operationen ermöglichen gleichzeitige Phasenübergänge für ganze Gruppen.
