# Sensorik

Kamerplanter unterstützt vier Datenquellen mit automatischer Fallback-Kette: automatisch (IoT/MQTT) → halbautomatisch (Home Assistant REST API) → Wetter-API (DWD/Open-Meteo für Freiland) → manuelle Eingabe. Die Datenherkunft wird immer protokolliert.
