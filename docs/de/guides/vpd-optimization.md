# VPD-Optimierung

Das Dampfdruckdefizit (VPD — Vapor Pressure Deficit) beschreibt den Unterschied zwischen dem aktuellen Wasserdampfdruck in der Luft und dem maximalen Dampfdruck bei gegebener Temperatur. Es ist ein entscheidender Parameter für Transpiration und Nährstoffaufnahme.

!!! note "Platzhalter"
    Dieser Inhalt wird in einem folgenden Schritt ausgearbeitet.

## Zielwerte nach Phase

| Phase | VPD-Ziel (kPa) |
|-------|----------------|
| Keimung / Sämling | 0,4 – 0,8 |
| Vegetativ | 0,8 – 1,2 |
| Blüte | 1,0 – 1,5 |
