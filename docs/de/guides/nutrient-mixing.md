# Nährlösung mischen

Eine korrekt gemischte Nährlösung ist entscheidend für optimales Pflanzenwachstum. Kamerplanter berechnet automatisch EC-Budgets und validiert die Mischfolge.

!!! danger "Mischfolge einhalten"
    CalMag **immer zuerst** in Wasser einrühren, bevor andere Dünger hinzugefügt werden — besonders vor Sulfaten. Falsche Reihenfolge führt zu Ausfällungen und unwirksamer Nährlösung.

!!! note "Platzhalter"
    Dieser Inhalt wird in einem folgenden Schritt ausgearbeitet.

## EC-Budget

```
EC_netto = EC_ziel - EC_grundwasser
```

Kamerplanter berücksichtigt automatisch den EC-Wert der Wasserquelle (Leitungswasser oder RO-Wasser).
