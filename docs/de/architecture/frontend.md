# Frontend-Architektur

Das Frontend ist in React 19 mit TypeScript 5.9 (strict) implementiert.

!!! note "Platzhalter"
    Dieser Inhalt wird in einem folgenden Schritt ausgearbeitet.

## Tech Stack

- **React 19** + **TypeScript 5.9** (strict)
- **MUI 7** (Material UI) als Komponentenbibliothek
- **Redux Toolkit** für State Management
- **react-router-dom v7** für Routing
- **react-i18next** für i18n (DE/EN)
- **Vite 6** als Build-Tool
