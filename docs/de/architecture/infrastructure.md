# Infrastruktur

Kamerplanter wird auf Kubernetes betrieben und mit Helm und Skaffold verwaltet.

!!! note "Platzhalter"
    Dieser Inhalt wird in einem folgenden Schritt ausgearbeitet.

## Komponenten

- **Kubernetes 1.28+** — Orchestrierung
- **Helm** — Paketmanagement (Charts unter `helm/`)
- **Traefik** — Ingress Controller
- **Skaffold** — Entwicklungsworkflow
