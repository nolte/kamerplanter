# Debugging

Anleitungen zum Debuggen des Backends und Frontends in der lokalen Entwicklungsumgebung.

!!! note "Platzhalter"
    Dieser Inhalt wird in einem folgenden Schritt ausgearbeitet.

## Backend (debugpy)

```bash
skaffold debug --port-forward
```

Verbinde einen Python-Debugger auf Port `5678`.

## Frontend

Browser DevTools und React Developer Tools reichen für die meisten Frontend-Diagnosen.
