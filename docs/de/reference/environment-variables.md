# Umgebungsvariablen

Konfigurationsparameter für das Backend.

!!! note "Platzhalter"
    Dieser Inhalt wird in einem folgenden Schritt ausgearbeitet.

## Pflichtfelder

| Variable | Beschreibung | Beispiel |
|----------|-------------|---------|
| `ARANGO_HOST` | ArangoDB-Host | `localhost` |
| `ARANGO_PORT` | ArangoDB-Port | `8529` |
| `ARANGO_DB` | Datenbankname | `kamerplanter` |
| `ARANGO_USER` | Datenbanknutzer | `root` |
| `ARANGO_PASSWORD` | Datenbankpasswort | — |
| `REDIS_URL` | Redis-Verbindungs-URL | `redis://localhost:6379/0` |
| `JWT_SECRET_KEY` | Schlüssel für JWT-Signierung | — |
