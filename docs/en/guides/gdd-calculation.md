# GDD Calculation

Growing Degree Days (GDD) measure the accumulated heat a plant has experienced since sowing. They allow more reliable predictions of harvest time and phase transitions than calendar time alone.

## Formula

```
GDD = max(0, (T_max + T_min) / 2 - T_base)
```

- `T_max` — Daily maximum temperature (°C)
- `T_min` — Daily minimum temperature (°C)
- `T_base` — Plant base temperature (°C)
